# 项目结论

## 一、项目结果概览

本项目围绕“下季度是否继续在读”构建学生流失预警方案，并将模型结果转化为业务可执行的跟进优先级。核心结果如下：

1. 样本量为 5,527，真实流失样本为 756，整体流失率为 13.68%。
2. 模型 Accuracy 为 0.7055，Recall 为 0.7460，AUC 为 0.8031。
3. 高风险学生 303 人，占整体 5.48%，其中真实流失占比达到 73.27%。
4. 风险标签主要集中在互动弱风险，说明续读流失更多是“连接变弱”而不是单一价格因素。

## 二、核心业务发现

### 1. 高风险名单具备明显筛选价值

- 整体流失率只有 13.68%，但高风险分层中的真实流失占比达到 73.27%。
- 这说明模型输出的高风险名单可以直接作为老师和运营的重点触达池。

### 2. 互动信号是最重要的先行指标

- 原因标签统计中，互动弱风险及其组合标签占比最高。
- 说明好友关系、互动频次、家长主动沟通等行为信号，对续读判断具有很强参考价值。

### 3. 业务动作不能只盯流失概率

- 同样是高流失概率学生，有的人可挽回空间更大，有的人业务价值更高。
- 因此项目引入 `干预优先级分数`，把“风险高不高”“值不值得救”“能不能救”三个问题合并考虑。

## 三、可落地建议

1. 将高风险名单作为重点跟进池，由老师或课程顾问优先触达。
2. 对互动弱风险学生优先补学习反馈、恢复沟通链路，再判断续读意向。
3. 对服务响应风险学生检查老师回复效率，避免因体验问题造成非必要流失。
4. 在排班或人力有限的情况下，按 `干预优先级分数` 而不是只按流失概率排序。
