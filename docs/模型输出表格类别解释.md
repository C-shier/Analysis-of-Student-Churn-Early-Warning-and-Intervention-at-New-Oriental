# 模型输出表格类别解释

本文档用于解释 `新东方学生流失预警模型输出汇总表.xlsx` 中各类表格、风险类别、原因标签和建议动作的含义。

## 一、工作表类别解释

| 工作表名称 | 作用说明 |
|---|---|
| `模型评估总览` | 展示模型整体效果，包括样本量、流失率、Accuracy、Precision、Recall、F1、AUC 等指标。 |
| `风险等级分布` | 统计高风险、中风险、低风险分别有多少学生，以及各风险等级占比。 |
| `风险命中情况` | 对比不同风险等级下真实流失和真实未流失的人数，用于判断预警名单是否有效。 |
| `原因标签统计` | 统计每类流失原因标签出现的次数和占比，帮助判断主要风险来源。 |
| `Top50干预名单` | 按 `干预优先级分数` 从高到低排序，列出最需要优先跟进的 50 个学生。 |
| `字段说明` | 解释结果表中每个字段的业务含义。 |
| `完整预警明细` | 包含所有 5527 条学生记录的完整模型输出结果。 |

## 二、风险等级类别解释

风险等级由模型预测出的 `流失概率` 划分。

| 风险等级 | 判断标准 | 业务含义 | 建议处理方式 |
|---|---|---|---|
| 高风险 | 流失概率 >= 0.70 | 模型认为该学生下季度流失可能性较高。 | 优先进入老师或课程顾问重点跟进名单。 |
| 中风险 | 0.40 <= 流失概率 < 0.70 | 学生存在一定流失风险，但风险程度低于高风险学生。 | 加强课堂互动和学习反馈，持续观察后续变化。 |
| 低风险 | 流失概率 < 0.40 | 学生流失风险较低。 | 正常维护，定期观察即可。 |

在实际使用中，建议优先筛选 `风险等级 = 高风险` 的学生，再按照 `干预优先级分数` 从高到低排序。

## 三、流失原因标签类别解释

`流失原因标签` 用于解释学生为什么会被模型预警。一个学生可能同时存在多个原因标签。

| 原因标签 | 含义解释 | 可能对应的数据表现 |
|---|---|---|
| 互动弱风险 | 学生与老师或运营之间互动不足，学习连接较弱。 | 未加好友、当天无互动、聊天次数低。 |
| 服务响应风险 | 老师回复较慢，可能影响学生或家长的服务体验。 | 老师回复平均时长较长。 |
| 购买意愿弱风险 | 学生继续购买或续读意愿可能不足。 | 非联报、实际缴费课次较低。 |
| 新生适应风险 | 学生可能尚未形成稳定学习习惯或对课程体系不熟悉。 | 新生、上季度不在读。 |
| 综合模型预警 | 没有明显单一原因，但模型综合判断存在流失风险。 | 多个特征综合后显示风险较高。 |

常见组合标签示例：

| 组合标签 | 解释 |
|---|---|
| 互动弱风险、购买意愿弱风险 | 学生既缺少互动，又可能续报意愿不足。 |
| 服务响应风险、购买意愿弱风险 | 学生可能同时受到服务体验和购买意愿影响。 |
| 互动弱风险、购买意愿弱风险、新生适应风险 | 学生互动弱、购买意愿不足，同时又存在新生适应问题，需要重点关注。 |

## 四、建议动作类别解释

`建议动作` 是根据学生的风险等级和原因标签生成的运营跟进建议。

| 建议动作 | 适用对象 | 处理重点 |
|---|---|---|
| 当天电话/微信重点触达，补充学习反馈并确认续读意向 | 高风险且存在互动弱风险的学生。 | 先恢复沟通连接，再确认不续读原因。 |
| 检查老师响应链路，安排班主任跟进服务体验 | 存在服务响应风险的学生。 | 优先改善服务体验，减少因响应慢导致的流失。 |
| 列入重点挽回名单，由老师和课程顾问联合跟进 | 高风险且需要重点挽回的学生。 | 由教学和销售/顾问共同推进续读沟通。 |
| 加强课堂互动和学习反馈，观察后续沟通变化 | 中风险学生。 | 通过学习反馈和互动提升学生黏性。 |
| 正常维护，持续观察互动和缴费变化 | 低风险学生。 | 保持常规服务，不需要过度干预。 |

## 五、核心字段类别解释

| 字段 | 类别 | 解释 |
|---|---|---|
| `流失概率` | 模型预测结果 | 学生下季度流失的概率，越高表示越可能流失。 |
| `风险等级` | 风险分层结果 | 将流失概率转化为高、中、低三个业务可读等级。 |
| `可挽回系数` | 运营可触达指标 | 衡量该学生是否仍有沟通基础和挽回空间。 |
| `学员价值系数` | 业务价值指标 | 衡量该学生对续读和运营的价值。 |
| `干预优先级分数` | 最终排序指标 | 综合流失概率、可挽回性和学员价值，用于决定优先跟进顺序。 |
| `流失原因标签` | 可解释性结果 | 给出该学生被预警的主要原因。 |
| `建议动作` | 运营落地结果 | 给出老师或运营人员可以采取的后续动作。 |

其中，最建议重点关注的是：

```text
风险等级
流失概率
干预优先级分数
流失原因标签
建议动作
```

这几个字段可以直接支持业务跟进。

## 六、业务决策指标详细解释

`可挽回系数`、`学员价值系数` 和 `干预优先级分数` 是本方案设计的三个业务决策指标，它们不是原始 Excel 中自带的字段，而是为了让模型结果更容易落地使用而新增的指标。

### 1. 可挽回系数

`可挽回系数` 用来判断学生虽然存在流失风险，但是否仍然有机会被老师或运营人员挽回。

简单理解：

```text
可挽回系数越高 = 学生还有沟通基础，仍然有较大挽回机会
可挽回系数越低 = 学生互动较弱，触达难度较高，挽回难度更大
```

该指标主要参考以下字段：

| 参考字段 | 解释 |
|---|---|
| `是否好友 0否 1是` | 已经添加好友，说明可以直接触达。 |
| `当天是否互动` | 当天发生互动，说明学生或家长仍有回应。 |
| `聊天次数` | 聊天次数越多，说明沟通基础越好。 |
| `客户主动聊天次数` | 家长或学生主动联系，说明仍有参与意愿。 |
| `是否优质聊天` | 沟通质量较好，说明存在较好的挽回空间。 |

例如，一个学生流失概率很高，但 `可挽回系数` 也很高，说明他虽然有流失风险，但仍能联系、仍有沟通基础，应该优先跟进。

### 2. 学员价值系数

`学员价值系数` 用来判断该学生是否值得重点维护和优先挽回。

简单理解：

```text
学员价值系数越高 = 学生购买承诺、续读基础或长期维护价值越高
学员价值系数越低 = 学生当前业务价值或续读基础相对较弱
```

该指标主要参考以下字段：

| 参考字段 | 解释 |
|---|---|
| `实际缴费课次` | 缴费课次越多，说明购买承诺越强。 |
| `联报标签` | 联报学生通常代表更强的购买意愿和续读基础。 |
| `上季度常规是否在读` | 上季度在读说明学生有连续学习基础。 |
| `生源类型(新老生标签)` | 老生通常具有更高的长期维护价值。 |

需要注意的是，`联报标签` 和 `联报类型` 没有进入预测模型，以避免影响对流失概率的判断；但它们可以在运营排序阶段作为学生价值参考。

### 3. 干预优先级分数

`干预优先级分数` 是最终用于排序的核心业务指标。

计算逻辑为：

```text
干预优先级分数 = 流失概率 × 可挽回系数 × 学员价值系数
```

它综合回答三个问题：

| 问题 | 对应指标 |
|---|---|
| 这个学生流失风险高不高？ | `流失概率` |
| 这个学生还有没有机会挽回？ | `可挽回系数` |
| 这个学生值不值得优先挽回？ | `学员价值系数` |

因此，实际业务中不建议只按 `流失概率` 排序，而应优先按 `干预优先级分数` 排序。分数越高，说明该学生越应该被优先安排老师或课程顾问跟进。

举例说明：

| 情况 | 解读 |
|---|---|
| 流失概率高、可挽回系数高、学员价值系数高 | 最优先跟进，属于高风险且值得挽回的学生。 |
| 流失概率高、可挽回系数低 | 虽然风险高，但触达难度较大，需要更强触达方式。 |
| 流失概率中等、可挽回系数高 | 适合提前干预，可能用较低成本降低流失风险。 |
| 流失概率低 | 正常维护即可，不需要占用重点跟进资源。 |

这三个指标的价值在于：它们把模型结果从“谁可能流失”进一步转化为“谁最值得优先跟进”。

## 七、如何使用这份表格

实际业务使用时，可以按以下顺序查看：

1. 先看 `模型评估总览`，了解模型整体效果。
2. 再看 `风险等级分布`，了解高、中、低风险学生规模。
3. 查看 `风险命中情况`，判断高风险名单是否有较强参考价值。
4. 查看 `原因标签统计`，判断主要流失原因集中在哪些方面。
5. 最后查看 `Top50干预名单`，优先安排老师或课程顾问跟进。

如果需要查看所有学生，则进入 `完整预警明细`，根据 `风险等级`、`科目`、`年级序数处理值`、`流失原因标签` 等字段进行筛选。

## 八、总结

该汇总表不是单纯展示模型预测结果，而是把模型结果转化为可执行的业务表格。它能帮助业务人员快速判断：

1. 哪些学生最可能流失。
2. 哪些学生最值得优先挽回。
3. 学生被预警的主要原因是什么。
4. 老师或运营人员应该采取什么跟进动作。
